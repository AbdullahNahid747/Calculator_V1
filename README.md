# 🚀 Smart Calculator

A simple yet powerful command-line calculator built with Python. It supports basic arithmetic operations, square roots, powers, and keeps track of calculation history.

## ✨ Features

* ➕ Addition
* ➖ Subtraction
* ✖ Multiplication
* ➗ Division
* 🔢 Powers (`2 ^ 5`)
* √ Square Root (`sqrt(25)`)
* 📜 Calculation History
* ⚡ Fast and Lightweight
* 🐍 Built with Python

## 🚀 Getting Started

### Prerequisites

* Python 3.8 or higher

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/smart-calculator.git
```

Navigate to the project folder:

```bash
cd smart-calculator
```

Run the application:

```bash
python calculator.py
```

## 📖 Usage

Examples:

```text
10 + 5
8 * 7
2 ^ 5
100 / 4
sqrt(25)
```

Commands:

```text
history   # View calculation history
exit      # Exit the program
```
